# AT-UI React Examples

This folder contains a complete example application showcasing all AT-UI React components.

## Getting Started

### Installation

```bash
cd examples
npm install
```

### Development

```bash
npm run dev
```

This will start a development server at `http://localhost:3000` with hot module replacement.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

This example app demonstrates:

- **Form Components**: Input, Textarea, Checkbox, Radio, Switch, Slider
- **Display Components**: Tag, Badge, Alert, Progress, Rate
- **Layout Components**: Card, Collapse, Tabs, Steps, Timeline, Breadcrumb, Pagination
- **Interactive Components**: Tooltip, Popover, Modal
- **Feedback Components**: Message, Notification, LoadingBar

All components are fully interactive and demonstrate their key features and props.

## Component Usage

Each component section shows:
- Basic usage examples
- Different variants and types
- Interactive demos
- Common props and configurations

## Notes

- Make sure the parent `at-ui` package is built before running examples
- Some components may require additional styling from `at-ui-style`
- Placeholder components (Select, Dropdown, Menu, Table) show basic structure
